# BAHT RESIDENCE — Telegram Bot + Mini App

## Что это
- **Telegram Bot** — вы добавляете объекты командой `/add`, бот публикует их в канал
- **Mini App** — красивый каталог, который открывается прямо в Telegram
- **Хранилище** — все объекты сохраняются в файл `properties.json` на вашем сервере

---

## Шаг 1 — Создайте бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Напишите `/newbot`
3. Придумайте имя: `BAHT RESIDENCE`
4. Придумайте username: `bahtresidence_bot`
5. Скопируйте **токен** — он выглядит так: `1234567890:AAxxxxxx...`

---

## Шаг 2 — Добавьте бота в канал

1. Зайдите в ваш канал `@bahtresidence`
2. Настройки → Администраторы → Добавить администратора
3. Найдите вашего бота → дайте права **публикации сообщений**

---

## Шаг 3 — Узнайте ваш Telegram ID

1. Напишите [@userinfobot](https://t.me/userinfobot)
2. Он ответит вашим ID (число, например `123456789`)

---

## Шаг 4 — Деплой на Railway (бесплатно)

1. Зарегистрируйтесь на [railway.app](https://railway.app)
2. Нажмите **New Project → Deploy from GitHub**
3. Загрузите все файлы этой папки в GitHub репозиторий
4. В Railway добавьте переменные окружения:

| Переменная   | Значение                        |
|-------------|----------------------------------|
| BOT_TOKEN   | Токен от BotFather               |
| CHANNEL_ID  | @bahtresidence                   |
| ADMIN_IDS   | Ваш Telegram ID                  |
| MINIAPP_URL | URL проекта (Railway покажет)    |
| PORT        | 8080                             |

5. После деплоя скопируйте URL проекта (напр. `https://baht-bot.railway.app`)
6. Вставьте этот URL в переменную `MINIAPP_URL` и задеплойте снова

---

## Шаг 5 — Подключите Mini App к кнопке в канале

В [@BotFather](https://t.me/BotFather):
```
/mybots → ваш бот → Bot Settings → Menu Button
Введите URL: https://ваш-проект.railway.app
```

## Команды бота (для администраторов)

| Команда         | Что делает                          |
|----------------|--------------------------------------|
| `/add`          | Добавить новый объект                |
| `/list`         | Список всех активных объектов        |
| `/delete [ID]`  | Удалить объект (ID из /list)         |
| `/stats`        | Статистика каталога                  |
| `/help`         | Список команд                        |

---

## Как добавить объект

1. Напишите боту `/add`
2. Отправьте **фото** объекта
3. Введите **название** (напр. "Sky Residence")
4. Выберите **тип** (Новостройка / Вторичное / Аренда / Коммерция / Дом)
5. Введите **цену** (напр. "$150 000" или "$2 500/мес")
6. Введите **адрес** (напр. "Мирабадский р-н, ул. Навои, 45")
7. Введите **площадь** (напр. "95 м² · 3 комн")
8. Введите **описание** (или `-` чтобы пропустить)

Готово! Объект автоматически появится в канале и в каталоге Mini App.

## Локальный запуск (для тестирования)

```bash
# Установите Python 3.10+
pip install -r requirements.txt

# Создайте .env файл
cp .env.example .env
# Откройте .env и заполните свои значения

# Запустите
python bot.py
```
